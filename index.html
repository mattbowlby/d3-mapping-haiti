Introduction to D3.js Web Mapping Through 4 Simple Maps

This group of D3 maps of Haiti were inspired by a presentation by @rgdonohue for @maptimeLEX. The complete set of maps used in the presentation can be found on their GitHub page. The maptimeLEX maps draw upon Andy Woodruff's (@awoodruff) excellent @maptimeBoston presentation.

This repo was fully inspired by the above-mentioned work. All the files and data for the examples here are hosted atgithub.com/mattbowlby.
